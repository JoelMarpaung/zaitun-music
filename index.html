How it works

1

Load audio

Drop an audio file or paste a YouTube link. Files are checked against the 30 MB and 10 minute limits.

2

Estimate chords

The Zaitun chord system analyzes the audio and returns timestamped chord events.

3

Play along

The timeline syncs with playback, supports transposition, and offers beginner simplification.

Powered by the Zaitun chord system

Uploaded files and YouTube links are sent to the Zaitun chord system. The frontend expects chord events as JSON. 

{
  "key": "G",
  "chords": [
    { "time": 0, "duration": 2.4, "chord": "G" },
    { "time": 2.4, "duration": 2.4, "chord": "Em" }
  ]
}
